Slider

Introduction

The Slider-Plugin allows you to drag&drop a knob inside an element to set a value within a certain range.

Use the slider to change the font size.

Change the value, to change the fontsize.


Why? This is a nice User-Interface-Element to not just let the user set a value by typing into an input field but limiting the range without further needs to check the user input (on clientside). With callbacks you can manipulate anything after changing the value.
